# github-notes

Git= Version control software which means it keeps track of the history of what your working on. It also allows a lot of people to work on a project together. 

Github= website or webserive where you CAN have an account and do git things in a public server. It runs git.

Repo= a project where it can have a lot of files with it. You can name it. A README is like a description. 

commit= save or change the file and save that changed file. You can name each commit which will help you save points. You can check each time you commit. plus and green means added. Red and minus means removed in a new commit.
